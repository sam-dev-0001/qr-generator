Wi-Fi QR Code Generator

Print it by the router or on a welcome card: guests scan, tap connect, and they're online — no password typing, no spelling out special characters.

Create a Wi-Fi QR code

Where this saves the most hassle

Cafés, short-term rentals, waiting rooms and home guest rooms are the classic spots. It's especially useful for long, random passwords that are painful to type on a phone keyboard.

Security note

The password is encoded directly into the QR image. Anyone who scans it can join your network, so treat a printed Wi-Fi QR code the way you'd treat the password itself — don't post it somewhere public if the network is private.

Frequently asked questions

What if my network has no password?

Choose "None (open network)" as the security type and leave the password field blank.

Does it support hidden networks?

Yes — tick the "Hidden network" option so connecting devices know to search for it by name.

Which security types are supported?

WPA/WPA2/WPA3, WEP, and open (no password) networks.
