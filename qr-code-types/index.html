QR Code Types

Every kind of QR code QRGenPro can generate, with guidance on when to use each one and how to format the content for a clean scan.

Text QR Code

Encode a note, a quote, a serial number, or any block of plain text up to 1,500 characters into a QR code that any phone camera can scan instantly.

URL QR Code

Point a scan straight at a website — a landing page, a menu, a portfolio, an app store listing — with a QR code you can brand to match.

Email QR Code

Scan-to-email: the code opens the visitor's mail app with your address, and optionally a subject line and message, already filled in.

Phone QR Code

One scan dials your number. Great for business cards, van signage, service stickers and anywhere a fast call beats typing digits.

SMS QR Code

Scan to open a pre-addressed text message, with an optional pre-written line — handy for opt-ins, RSVPs and quick support requests.

Wi-Fi QR Code

Print it by the router or on a welcome card: guests scan, tap connect, and they're online — no password typing, no spelling out special characters.

vCard / Contact QR Code

A vCard QR code lets someone scan and immediately save your name, phone, email and company straight into their contacts app — no typing.

PDF QR Code

A QR code can't contain a whole PDF file, but it can point straight at one — scan and the document opens or downloads on the visitor's phone.

Event QR Code

Scan and add straight to the calendar app — no manually typing the date, time and location off an invite or poster.

Location QR Code

Scan and jump straight into a maps app pointed at an address or a precise set of coordinates — perfect for venues, trailheads and delivery drop-offs.
