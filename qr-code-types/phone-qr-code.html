One scan dials your number. Great for business cards, van signage, service stickers and anywhere a fast call beats typing digits.

Create a phone QR code

Where click-to-call codes shine

Field-service vehicles, appliance repair stickers, real-estate yard signs and after-hours support numbers all benefit — the person scanning doesn't need to save a contact or dial manually.

Formatting notes

  • Include the country code (e.g. +1) if the code may be scanned internationally.
  • Spaces, dashes and parentheses are fine — they're stripped automatically when the code is generated.

Frequently asked questions

What format should the number be in?

Any common format works, such as +1 555 123 4567 — formatting characters are removed automatically.

Does it work on both iPhone and Android?

Yes, both use the standard tel: link that opens the native phone dialer.
