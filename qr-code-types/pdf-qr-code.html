PDF QR Code

A QR code can't contain a whole PDF file, but it can point straight at one — scan and the document opens or downloads on the visitor's phone.

Build a PDF-link QR code

How it actually works

Upload your PDF somewhere it can be reached by a public link — your website, Google Drive, Dropbox, or a cloud storage bucket — then generate a URL QR code pointing at that link. Scanning opens the file in the phone's browser or PDF viewer.

Good uses

  • Restaurant menus, spec sheets, event programs, instruction manuals, resumes.
  • Keep the file size reasonable so it opens quickly on mobile data.
  • If the PDF changes often, host it at a fixed link so the same QR code keeps working.

Frequently asked questions

Can a QR code store the PDF itself?

No — QR codes have a strict data capacity, far too small for a real document. It stores a link to the file instead.

Where should I host the PDF?

Anywhere with a stable public link works: your own site, Google Drive, Dropbox, or similar.
