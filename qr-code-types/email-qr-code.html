Email QR Code Generator

Scan-to-email: the code opens the visitor's mail app with your address, and optionally a subject line and message, already filled in.

Create an email QR code

Good uses for an email QR code

Event badges, support desks, feedback cards and press kits all benefit from a one-scan way to start an email — no typing an address by hand, no typos.

Setting it up well

  • Pre-fill the subject line so replies are easy to route (e.g. "Event Feedback").
  • Keep the message short; it's a starting point the sender can edit.
  • Double-check the address — this generator validates the format but not deliverability.

Frequently asked questions

Will it send the email automatically?

No. It opens the recipient's default mail app with the fields prefilled — the sender still has to hit send.

Can I leave the subject and message blank?

Yes, both are optional; only the email address is required.

Does this work with Gmail, Outlook and Apple Mail?

Yes — it uses the standard mailto: format supported by every major mail client.
